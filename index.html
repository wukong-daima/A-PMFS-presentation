03 · Hypothesis

핵심 가설

H1. Target 카드와 벡터 유사도가 가까운 한국어 키워드 프라이밍이 LLM 페르소나의 카드 선택을 baseline 대비 유의미하게 lift시킨다.
H2. 프라이밍은 단계적(color → suit → full)으로 누적되며, hit rate는 단조 증가 패턴을 보인다.
H3. Placebo (의미적 무관 프라이밍)는 lift 효과를 만들지 못한다 — 효과는 단순 "텍스트 노출"이 아닌 "의미적 가까움" 때문이다.
04 · Glossary

용어 & 약어 정리

마술·심리학

ACAAN
Any Card At Any Number — 관객이 지정한 카드가 지정한 위치에서 나오는 멘탈리즘의 대표 효과.
Mental Forcing
관객이 자유롭게 선택했다고 믿게 하면서 마술사가 카드를 유도하는 기술.
Priming (프라이밍)
먼저 접한 자극이 후속 판단·반응에 무의식적으로 영향을 주는 심리 현상.
Darren Brown 3-level
색(color) → 문양(suit) → 숫자(rank) 단계적 priming 기법.

통계·기술

Embedding
텍스트를 의미를 보존하는 고차원 실수 벡터(예: 3072차원)로 변환.
Cosine similarity
두 벡터 사이의 각도로 의미적 가까움을 측정 (−1 ≤ sim ≤ 1).
Ablation
구성 요소를 단계적으로 추가/제거하며 효과를 분리 측정.
χ² (chi-square)
관측 빈도와 기대 빈도의 차이를 검정하는 통계량.
Cramér's V
χ² 기반의 표본 크기 독립 효과 크기 지표 (0~1).
Wilson 95% CI
이항 비율의 신뢰구간 — 작은 표본에서도 강건.
05 · Design

5-arm Ablation 설계

"Arm"이란? Treatment Arm의 줄임 — 임상시험·실험설계에서 동일 모집단을 서로 다른 처치 조건에 배정하는 그룹을 가리키는 표준 용어. 5-arm은 5개의 비교 그룹을 의미합니다.
Arm이름처치목적
AControl프라이밍 없음자연 baseline
BColor색상 단계 프라이밍1단계 효과
CColor + Suit색+문양 프라이밍2단계 효과
DFull색+문양+숫자 모든 단계최대 효과
EPlacebo무관 텍스트 프라이밍"순수 노출 효과" 제거
왜 5-arm: A는 baseline, B/C/D는 priming 단계 비교, E는 placebo control. Per-persona × per-arm 디자인으로 개인차(individual variance) 통제.

12 · Embedding

Embedding & Vector 유사도

키워드 추출 + 정렬 메커니즘

1. target_card = "다이아_7"
2. GPT-4o-mini → 50개 한국어 키워드 후보 추출
   (의미·시각·정서·문화적 연관)
3. text-embedding-3-large로 임베딩
   keyword_vec ∈ ℝ^3072
4. cosine_sim(target_card_vec, keyword_vec_i)
   = (v_t · v_k) / (‖v_t‖·‖v_k‖)
5. 정렬 후 Top-N 선정 (level별 5/10/15)
6. priming 스크립트로 직접 노출

왜 효과적인가

  • 판단(judge) 노이즈 제거 — sim에 90% 가중치
  • 키워드가 카드의 "의미적 좌표"를 직접 전달
  • 메타포는 의미가 우회되어 희석됨

14 · Statistics

통계 검정 Framework

사용한 검정 방식

  • Per-arm hit rate
    + Wilson 95% 신뢰구간
  • Pairwise χ² test
    (2×2 contingency, A vs B/C/D/E)
  • Cramér's V
    effect size (sample size 독립)
  • Omnibus χ² (5-arm)
    overall difference 검정
  • vs Uniform (1/52)
    random baseline 대비
유의 수준 α = 0.05
(★ p<0.05, ★★ p<0.01, ★★★ p<0.001)

각 방식의 의미

χ² (chi-square) test
관측 빈도와 기대 빈도의 차이를 정량화. 두 그룹의 hit/miss 분포가 우연히 같은지 검정. p-value가 작을수록 우연 가능성이 낮음.
Pairwise vs Omnibus
Pairwise는 두 arm씩 비교(A vs D 등), Omnibus는 5개 arm 전체가 동일한지 한 번에 검정. 다중 비교 보정 효과.
Cramér's V (효과 크기)
표본이 커지면 p-value는 자연스레 작아지므로, 효과의 실질적 크기를 별도 측정. 0.1 small · 0.3 medium · 0.5 large 기준.
Wilson 95% CI
이항 비율의 신뢰구간. 표본을 다시 뽑아도 95% 확률로 이 구간 안. 작은 hit rate(예: 1%)에서도 강건.
p-value
귀무가설(차이 없음)이 참일 때 관측 결과 이상의 차이가 우연히 나타날 확률. α=0.05 미만이면 "통계적으로 유의".

16 · Diagnosis

진단 — 왜 효과가 없었나?

1. Target 선택 문제

  • 자연 Top-1은 baseline ≈ 44% → ceiling effect
  • → unfavored target으로 재설계 (baseline ~0.5%)

2. 프라이밍 선정 가중치 문제

  • 0.3·sim + 0.7·judge → judge가 sim 무시
  • K=10 중 sim 최저 후보가 selected (예: best=0.156 < others mean=0.179)
  • → 가중치 0.9·sim + 0.1·judge로 조정

3. 프라이밍 형식 문제 (핵심)

  • Brown 메타포는 "의미 우회" — target과 sim 0.24–0.27 정체
  • → 키워드 직접 노출 방식으로 재설계

읽는 법: 막대 높이 = arm별 hit rate (target 카드를 정확히 선택한 비율). 노란 점선은 B → C → D 단조 증가 추세. 의미: 1단계(color) → 2단계(+suit) → 3단계(+rank) 누적 priming이 hit rate를 단계적으로 끌어올림. 가설 H2 관측. 각 단계의 marginal lift는 1단계가 가장 크고(+1.30p) 추가 단계가 +0.33~0.5p씩 더해짐 — diminishing returns 패턴.
20 · H3 Result

H3 검증 — Placebo Control

플라시보 검증이란? 임상시험에서 가짜 약(설탕 알약)을 진짜 약과 비교해 효과의 진정성을 검증하는 방법. 본 연구에서는 의미적으로 무관한 텍스트를 진짜 priming과 동일한 형식·길이로 노출. 만약 placebo도 효과를 보이면 "그냥 텍스트만 봐도 카드 선택이 바뀐다"는 뜻이 되어 진짜 priming의 효과를 단정할 수 없습니다.

결과의 의미

  • "단순 텍스트 노출" 효과 ❌
  • "의미적 관련성" 효과 ✓
  • 가설 H3 강하게 지지
  • 임상시험 placebo control과 동등한 견고함

왜 ns가 좋은 결과인가

  • Placebo와 control이 통계적으로 같음 = priming의 효과가 의미에서 나옴이 분리 검증됨
  • 만약 E도 유의했으면 H1·H2의 결과가 priming 자체가 아닌 "텍스트 자극" 때문일 가능성

23 · Implications

의미와 한계

의미 (Contribution)

  • LLM에서 vector-aligned 프라이밍 효과 정량화
  • 5-arm Ablation으로 baseline·placebo 명확 분리
  • Cramér's V로 effect size 측정 (small but robust)
  • 재현 가능한 pipeline + parquet 데이터
  • Open-source 공개 (GitHub)

학술·실무적 시사점

  • LLM의 응답이 "의미적 인접 자극"에 영향받음을 통계적으로 입증
  • Prompt engineering의 priming 메커니즘 정량 분석 도구 제공
  • 인간 priming 실험의 가설 후보를 LLM으로 미리 screening 가능

한계 (Limitations)

  • LLM (GPT-4o) 한정 — 인간 일반화 불명
  • Effect size 0.082 (small) — 큰 lift는 아님
  • Per-archetype/mood breakdown은 더 큰 n 필요
  • Calibration 1,000 trials는 페르소나당 10회로 노이즈 가능
  • 한국어만 검증

외부 타당도 위협

  • LLM의 학습 데이터 분포에 의존 — 모델 업데이트 시 결과 변동 가능
  • 실제 마술 무대(인간 관객)에 직접 적용 불가
  • "hit rate"는 single-shot 측정 — 멀티턴 대화 priming은 미검증
